November 2017

Final Project in Python

Dietician Program

In our final class project for Python, we had to make a dietician program. So me and one of my classmate made a project that asks the user about his age, weight, height and his level of physical activity. The program then tells the user how many calories he should consume in a day. The program then gives the user suggested food items that the user should eat during breakfast, lunch and dinner in order to maintain the required daily calorie intake. All of this data is stored in a CSV file which can be edited later from the program.
